## Goals

- Practise working with real applications developed, maintained, and used by open source developers
- Attain high-level understanding of the Android development stack
- Become familiar with the basic mechanics of application deployment
- See first-hand what can be accomplished with the techqniques we learn this semester.

## Instructions

1. Create a [fork](#) of rhte [Now in Android](https://github.com/android/nowinandroid) application.
2. [Clone](#) the fork to your development environment.
3. Follow the [README](https://github.com/android/nowinandroid/blob/main/README.md) instructions to [build](#) the application.

[Build and run your app](https://developer.android.com/studio/run)
[Sign your app](https://developer.android.com/studio/publish/app-signing)
[Set up continuous integration](https://developer.android.com/studio/projects/continuous-integration)
